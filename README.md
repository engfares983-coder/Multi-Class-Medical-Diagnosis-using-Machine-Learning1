🩺 Multi-Class Medical Diagnosis using Machine Learning
An interactive web application built with Streamlit that leverages Machine Learning algorithms to predict and diagnose various diseases based on user-input symptoms.
🧠 Project Overview
This project aims to bridge the gap between complex data science models and end-users. By using a multi-class classification approach, the system can identify potential health conditions from a dataset of symptoms.
🚀 Key Features
ML-Powered Predictions: Uses advanced classification algorithms (Scikit-Learn).
User-Friendly Interface: Simple and clean UI built with Streamlit.
Data-Driven: Trained on a structured medical dataset (disease_diagnosis.csv).
Real-time Results: Instant diagnosis based on selected symptoms.
🛠️ Tech Stack
Language: Python 🐍
Machine Learning: Scikit-Learn (Classification models).
Data Analysis: Pandas, NumPy.
Web Framework: Streamlit.
📋 How It Works
Data Processing: The model processes medical data from disease_diagnosis.csv.
Model Training: A machine learning model is trained to recognize patterns between symptoms and specific diseases.
Inference: When a user selects symptoms in the app, the model calculates the most likely diagnosis.
